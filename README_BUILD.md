# VerseWatch Android Build (The Pro Way)

Kanka, bu yöntemle tünel ve link dertlerini tamamen bitiriyoruz. GitHub'ın kendi sunucuları senin için tertemiz bir APK üretecek.

## Nasıl Yapılır?

1.  Yeni bir GitHub deposu (Repository) aç.
2.  Bu klasördeki tüm dosyaları o depoya yükle.
3.  Dosyaları yüklediğin an GitHub (Actions sekmesinde) otomatik olarak APK derlemeye başlayacak.
4.  Bittiğinde "Artifacts" kısmından APK dosyanı indirebilirsin.

**Not:** Bu yöntem en profesyonel ve hatasız yoldur.
